#  Most Frequent Plant ID (Java)

This repository contains a Java program that identifies the **most common plant ID**
from a list of plant sightings.

Each plant is represented by a unique integer.  
If multiple plant IDs have the same highest frequency, the program returns the **smallest ID** among them.

---

##  Problem Statement

Given a list of integers representing plant sightings:

- Count the frequency of each plant ID
- Print the plant ID that appears most frequently
- If there is a tie, choose the smallest plant ID
